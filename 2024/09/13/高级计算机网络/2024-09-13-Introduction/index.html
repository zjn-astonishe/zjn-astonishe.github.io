Introduction

The different views of Internet

nuts and bolts view

  • Billions of connected computing devices

    • host：end systems(中央的主机，终端系统)
    • running network apps at Internets’ edge(边缘网络的应用)

  • Packet switches：forward packets(转发数据包，chunks of data，数据包)

    • routers(路由器)
    • switches(交换机)

  • Communication links(通信链路)

    • fiber(光纤), copper(铜的双绞线), radio(无线电波), satellite(卫星通信)
    • transmission rate：bandwidth(带宽)

  • Networks

    • collection of devices, routers, links
    • managed by an organization

service view

  • Infrastructure(基础架构)
    • provides services to applications
      • Web, streaming video, multimedia teleconferencing(多媒体电话会议), email, games, e-commerce(电子商务), social media, interconnected applicances(互联应用)
    • provides programming interface to distributed applications(分布式应用的编程接口)
      • hooks(句柄，like socket)
        • allowing sending/receiving apps to connect to, use Internet transport service
      • provides service options, analogous to postal service(邮件服务)

Needed for Networking

Application view

  • Applications
  • Application Characteristics
  • User perceptions(用户感知)

Control view

  • Forwarding/Routing Decision(交换策略，路由策略)
  • Control
  • Monitoring(监控，如对整个网络的拥塞控制)
  • Error Reporting(差错控制)
  • Peer performance matching(流量控制)

Data View

  • How Applications are break down into segment/packets/Frame(应用数据如何划分为报文段、分组、帧)
  • How Segment/Packet/Frame are transfered(如何传输分割好的报文段、分组、帧)
  • Error Detection/Handling(差错检错和控制)

Network Protocols

  • computers(devices) rather than humans is the role of protocols

  • all communication activity in Internet governed by protocols

  • Protocols define

    • the format, order of message sent and received among network entities(所收发的信息的形式，顺序)
    • actions taken on msg transmission, receipt(收发时候采取的动作)
